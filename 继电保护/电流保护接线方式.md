#### 相间电流保护：
- 三相星型接线
- 两相星型接线

| 中性点接地方式   | 故障类型             | 三相星型接线 | 两相星型接线（A、C相） | 备注                                   |
| --------- | ---------------- | ------ | ------------ | ------------------------------------ |
| **直接接地**  | 三相短路             | √      | √            | 所有相均有短路电流，两相星型至少测到两相                 |
|           | 两相短路（AB、BC、CA）   | √      | √            | 任何两相短路必包含A或C相，故能动作                   |
|           | 两相接地短路（AB、BC、CA） | √      | √            | 同上，故障相电流大                            |
|           | 单相接地（A相）         | √      | √            | A相CT测到大电流                            |
|           | 单相接地（B相）         | √      | ×            | B相无CT，A、C相电流很小（仅零序），不动作              |
|           | 单相接地（C相）         | √      | √            | C相CT测到大电流                            |
| **非直接接地** | 三相短路             | √      | √            | 短路电流大，动作                             |
|           | 两相短路             | √      | √            | 同上                                   |
|           | 两相接地短路           | √      | √            | 同上                                   |
|           | 单相接地（任何相）        | ×      | ×            | 接地电流仅为电容电流（通常<10A），远小于整定值，保护不动作；仅发信号 |
