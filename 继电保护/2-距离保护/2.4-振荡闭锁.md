
电力系统振荡是指系统中并列运行的发电机之间失去同步，导致系统各节点的电压、电流及系统的功率发生周期性剧烈变化的现象。这种状态会对依靠测量阻抗来判别故障的距离保护产生严重干扰，极易导致保护误动。
### 1. 系统等效模型与基本假设

为了进行理论推导，通常将发生振荡的复杂电力系统简化为**双机等值系统**。
假设系统 M 和系统 N 通过一条输电线路 $Z_L$ 相连，保护装置安装在 M 侧母线处（线路起点）。

**定义参数如下：**
- $\dot{E}_M$：M 侧等效系统的电势。
- $\dot{E}_N$：N 侧等效系统的电势。
- $Z_M$：M 侧保护安装处背后的等效系统阻抗。
- $Z_N$：N 侧等效系统阻抗。
- $Z_L$：保护线路的阻抗。
- $Z_{\Sigma}$：系统总阻抗，即 $Z_{\Sigma} = Z_M + Z_L + Z_N$。
    
在纯振荡状态下（无短路故障），假设两侧电势幅值相等（$|\dot{E}_M| = |\dot{E}_N| = E$），且 $\dot{E}_M$ 超前 $\dot{E}_N$ 的相位角（即系统功角差）为 $\delta$。为了简化相量推导，取 $\dot{E}_N$ 为参考向量：
$$\dot{E}_N = E$$
$$\dot{E}_M = E e^{j\delta}$$
### 2. 测量阻抗的理论推导

保护安装处（M侧）测量到的电流 $\dot{I}_m$ 和电压 $\dot{U}_m$ 可由电路基本定律得出：
系统中的流过的总电流：
$$\dot{I}_m = \frac{\dot{E}_M - \dot{E}_N}{Z_{\Sigma}} = \frac{E e^{j\delta} - E}{Z_{\Sigma}}$$
保护安装处的测量电压：
$$\dot{U}_m = \dot{E}_M - \dot{I}_m Z_M$$
距离保护装置计算出的测量阻抗 $Z_m$ 为：
$$Z_m = \frac{\dot{U}_m}{\dot{I}_m} = \frac{\dot{E}_M}{\dot{I}_m} - Z_M$$
将 $\dot{I}_m$ 代入上式中的第一项：
$$\frac{\dot{E}_M}{\dot{I}_m} = \frac{E e^{j\delta}}{\frac{E(e^{j\delta} - 1)}{Z_{\Sigma}}} = Z_{\Sigma} \frac{e^{j\delta}}{e^{j\delta} - 1}$$
利用欧拉公式对 $\frac{e^{j\delta}}{e^{j\delta} - 1}$ 进行化简：
$$\frac{e^{j\delta}}{e^{j\delta} - 1} = \frac{e^{j\delta}(e^{-j\delta} - 1)}{(e^{j\delta} - 1)(e^{-j\delta} - 1)} = \frac{1 - e^{j\delta}}{2 - 2\cos\delta} = \frac{1 - \cos\delta - j\sin\delta}{2(1-\cos\delta)} = \frac{1}{2} - j \frac{\sin\delta}{2(1-\cos\delta)}$$
根据半角公式 $\sin\delta = 2\sin\frac{\delta}{2}\cos\frac{\delta}{2}$ 以及 $1-\cos\delta = 2\sin^2\frac{\delta}{2}$，可得：
$$\frac{1}{2} - j \frac{\sin\delta}{2(1-\cos\delta)} = \frac{1}{2} - j \frac{1}{2} \cot\frac{\delta}{2}$$
最终得到**振荡时测量阻抗的精确数学表达式**：
$$Z_m = \left( \frac{Z_{\Sigma}}{2} - Z_M \right) - j \frac{Z_{\Sigma}}{2} \cot\frac{\delta}{2}$$
