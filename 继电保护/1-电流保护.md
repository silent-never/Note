- **整定电流**——基频短路电流：
	- 电流互感器（CT）二次侧输出的是三相实际电流
	- 整定值是基于计算的基频短路电流$$ I_\mathrm{k} = \frac{E_\varphi}{Z_\Sigma}= \frac{K_k E_\varphi}{Z_\mathrm{s}+Z_\mathrm{L}} $$式中参数释义： 
		- $E_\varphi$：短路点处的相电压； 
		 - $Z_\mathrm{L}$：短路点与保护安装处之间的阻抗； 
		 - $Z_\mathrm{s}$：保护安装处到系统等效电源之间的阻抗； 
		 - $K_k$：短路类型系数，三相短路取 $K_k=1$，两相短路取 $K_k=\frac{\sqrt{3}}{2}$
	- 由正序等效定则，[[正序等效定则]]
	- 大致$Z_1\approx Z_2 <Z_0$， $E$为相电压，故有以下表格分析

| 故障类型       | 正序电流 $I_{a1}$            | 基频短路电流 $I_{k}$              | 备注  |
| ---------- | ------------------------ | --------------------------- | --- |
| 三项短路       | $\frac{E}{Z_1}$          | $I_{k}=I_{a1}$              | 最大  |
| 单相接地（A相）   | $\frac{E}{Z_1+Z_2+Z_0}$  | $I_{kA} = 3 * I_{a1}$       | 略小  |
| 两相短路（BC相）  | $\frac{E}{Z_1+Z_2}$      | $I_{kB} = \sqrt{3}* I_{a1}$ | 最小  |
| 两相接地（BC接地） | $\frac{E}{Z_1+Z_2//Z_0}$ | $I_kB$ 分量合成，一般大于正序          |     |
