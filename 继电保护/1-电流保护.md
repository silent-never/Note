## 一、单侧电源网络的电流保护：
- 基于本单侧电源网络分析，下图是故障电流幅值分布曲线![[1.0.1电流曲线示意图.png |493]]

### 1、 $\text{I}$段-电流速断保护
- 整定原则：
	- 躲过**保护范围外**发生故障在**检测元件处**产生的**最大短路电流**
	- 核心目标：保护范围外发生故障的最可能误动情况也不使其误动
- 可靠系数：$K_{rel}'$
	- 公式$$I_{set.2}'=K_{rel}'I_{k.B.max}$$
	- 1、由于阻抗参数的误差，实际短路电流可能大于计算值
	- 2、非周期变量未考虑在不对称分解的三序网络中，实际短路电流大于基频短路电流
	- 3、由于电流继电器的误差，实际动作电流比标定可能比整定值偏小
	- 4、考虑必要的裕度
- 动作时间：
	- 0s（瞬时）
- 保护范围：
	- 略小于“本线路全长“
	- 计算：实际短路电流=整定电流  的位置$$\frac{K_k E_\varphi}{Z_\mathrm{s}+\alpha Z_\mathrm{L}}=\frac{K_{rel}' E_\varphi}{Z_\mathrm{s.min}+Z_\mathrm{L}}$$得到：$$\alpha=\frac{K_k}{K_{rel}'}-\frac{K_{rel}'Z_s-K_k Z_{s.min}}{K_{rel}'Z_L}$$当短路电流最小，最不利于保护动作的情况下，$K_k=\frac{\sqrt{3}}{2}$，保护范围有最小值。
- 作用：
	- 主保护（快速）
- 灵敏度校验：
	- 在最不利动作的情况下（最小运行方式的两项短路）的保护范围应≥15%线路长度
### 二、$\text{II}$段-限时电流速断保护
- 整定原则：
	- 躲过**相邻线路**的 **$\text{I}$ 段动作电流 
	- 核心目标：在相邻次级保护未动作前不误动
- 可靠系数：$K_{rel}''$
	- 公式：$$	I_{set.2}''=K_{rel}''I_{set.1}$$
	- 考虑到非周期分类有所衰减，通过保护1、2的电流相同，误差影响相同，故$K_{rel}''$取略小于$K_{rel}'$
- 动作时间：
	- 0.3~0.6s，一个时限
- 保护范围：
	- 略小于“本线路全长 + 相邻线路Ⅰ段范围”
- 作用
	- 主保护（补偿Ⅰ段死区）
- 灵敏度校验：
	- 灵敏系数：**保护范围内**发生**金属性短路**时**最不利动作**的计算量/保护动作整定值
	- 公式$$K_{sen}=\frac{I_{k.min}}{I_{set}''}$$
### 三、$\text{III}$段-定时限过电流保护
- 整定原则：
	- 躲过最大**正常运行状态**的负荷电流
	- 核心目标：在任何正常运行状态下不误动
- 可靠系数：
	- 公式：$$ I_{set}''' = K_{rel}''' \frac{K_{Ms}}{K_{re}} I_{L.max} $$自启动系数：$K_{Ms}$ 电动机的自启动电流大于正常运行时的电流
	  返回系数：   $K_{re}$   确认正常运行状态下误动后可以返回
- 动作时间：
	- 阶梯时限![[1.0.1 阶梯时限示意图.png]]
- 保护范围：
	- 所有下级线路
- 作用：
	- 后备保护
### 附录备注：
- 保护范围外/相邻电路：
	- 均指次级线路？
- **整定电流**——基频短路电流：
	- 电流互感器（CT）二次侧输出的是三相实际电流
	- 整定值是基于计算的基频短路电流$$ I_\mathrm{k} = \frac{E_\varphi}{Z_\Sigma}= \frac{K_k E_\varphi}{Z_\mathrm{s}+Z_\mathrm{L}} $$式中参数释义： 
		- $E_\varphi$：短路点处的相电压； 
		 - $Z_\mathrm{L}$：短路点与保护安装处之间的阻抗； 
		 - $Z_\mathrm{s}$：保护安装处到系统等效电源之间的阻抗； 
		 - $K_k$：短路类型系数，三相短路取 $K_k=1$，两相短路取 $K_k=\frac{\sqrt{3}}{2}$
	- 由正序等效定则，[[正序等效定则]]
	- 大致$Z_1\approx Z_2 <Z_0$， $E$为相电压，故有以下表格分析

| 故障类型       | 正序电流 $I_{a1}$            | 基频短路电流 $I_{k}$              | 备注  |
| ---------- | ------------------------ | --------------------------- | --- |
| 三项短路       | $\frac{E}{Z_1}$          | $I_{k}=I_{a1}$              | 最大  |
| 单相接地（A相）   | $\frac{E}{Z_1+Z_2+Z_0}$  | $I_{kA} = 3 * I_{a1}$       | 略小  |
| 两相短路（BC相）  | $\frac{E}{Z_1+Z_2}$      | $I_{kB} = \sqrt{3}* I_{a1}$ | 最小  |
| 两相接地（BC接地） | $\frac{E}{Z_1+Z_2//Z_0}$ | $I_kB$ 分量合成，一般大于正序          |     |
