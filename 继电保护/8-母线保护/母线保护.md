- 利用相邻元件保护装置切除母线故障
	- 母线故障几率较低，而且由于母线支路较多，包含所有支路将结构复杂故障概率大
	- 在35kV及以下电压的母线，对保护快速性不高时一般不采用专门母线保护
- 专门的母线保护
	- 电流差动保护
		- 基尔霍夫定律，节点电流守恒 
		- 1、躲过外部短路故障的最大不平衡电流 $I_{r,\text{set}} = K_{\text{rel}} I_{\text{unb},\text{max}} = K_{\text{rel}} \times 0.1 I_{k,\text{max}} / n_{\text{TA}}$
		- 2、联接支路较多接线复杂，二次回路断线几率较大，防止误动作，起动电流大于任一联接支路的最大负荷电流 $I_{r,\text{set}} = K_{\text{rel}} I_{L,\text{max}} / n_{\text{TA}}$
		- 校验灵敏系数：$K_{\text{sen}} = \frac{I_{k,\text{min}}}{I_{r,\text{set}} n_{\text{TA}}}$，一般大于2
	- 电流相位比较式：
		- 正常运行时，节点电流守恒
			- 相位上看，至少一条支路的相位与其余支路相反
		- 发生故障时，所有电源支路向故障点供给短路电流，负荷支路电流几乎为零
			- 所有电流同相，电流基本上是同相位
		- 高阻抗型：
			- 分析：
				- 外部故障时，
				- 非故障支路电流不很大，TA不易饱和，
				- 故障电流支路为其他各电源支路电流之和，TA极易饱和，
				- 饱和时励磁阻抗很小，虽然一次电流很大，但是几乎全部流入励磁回路，二次电流很小
				- 若继电器内阻小，二次侧电流大，则故障电流支路极易饱和，差动继电器将流过很大的不平衡电流
			- 为避免误动，故采用内阻很大的电压型继电器，
			- 外部故障时：
				- 各支路TA都不饱和，则差动正常计算，和电流为零，不会误动
				- 故障支路TA饱和，二次支路无电流输出，其余支路不平衡电流将难以进入高阻抗差动继电器，被迫通过故障之路的电流互感器二次绕组流通
				- 差动继电器不会动作
			- 内部故障时：
				- 各支路二次电流都流入差动继电器，电压电流较大，保护动作
				- 但是差流回路可能产生危险的过电压，必须采用过电压保护回路来减小此过电压
		- 中阻抗型为：低阻抗与高阻抗的折中方案
- 双母线同时运行
	- 要有选择故障母线的能力
		- 固定连接方式：
		- TA1，TA2，TA3……均为1母线的支路的互感器，
			- 共同将二次侧接在一起，并联母线联络线的互感器+，接在继电器KD1上
		- TA1'，TA2'，TA3'……均为2母线的支路的互感器，共同将二次侧接在一起
			- 共同将二次侧接在一起，并联母线联络线的互感器-，接在继电器KD1上
		- KD1与KD2的差动接在KD3上
		- 外部故障时，当1母线的支路故障时，
			- 1母线非故障支路均为流入电流，2母线支路均为流入电流，联络线电流从2母线到1母线
			- TA1为流出电流，其二次侧电流为总母线其余各支路的和，故等于其余TAX的二次侧电流与联络线上的电流之和，电流不经过KD1
			- TAX'二次侧电流之和流过联络线互感器二次侧，电流不经过KD2
			- 差动KD3也不起作用
		- 内部故障时，当1母线故障，
			- 1母线支路均为流入电流，2母线支路均为流入电流，联络线电流从2母线到1母线
			- TAX的二次侧电流与联络线感应电流之和共同流过KD1
			- TAX'二次侧电流之和流过联络线互感器二次侧，电流不经过KD2
			- 差动KD3检测到KD1与KD2的电流差，起动
		- 故KD1、KD2各只反映各自母线，KD3反应任一母线的故障
		- 要求连接方式固定，不能支路在两母线间跳转，缺乏灵活性
	- 电流相位比较式差动
		- 1、2母线支路二次侧各并联到一起，通过差动KA联接，KA与相位比较继电器KP串联
		- 联络线二次侧，也接到KP上
		- 区外故障时，KA差动不起动
		- 区内故障时，KA差动必起动
		- 而KP通过判断联络线电流方向来判断哪个母线故障
