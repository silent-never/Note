- 闭锁式：（多采用）
	- 保护范围外短路，反方向故障发出闭锁信号，保护不动作
	- 保护准备起动后，等待几个毫秒收不到闭锁信号即动作
- 允许式：
	- 保护范围内故障，正方向故障发出允许跳闸信号

- 负序方向纵联保护
	- 负序电流从故障点流出
- 长期发信式闭锁保护，不需要起动1允许，一直发出闭锁信号
- 弱电源侧：
	- 由于背后电源太弱（或没有电源），流过 B 侧的短路电流极小，甚至达不到 B 侧保护的启动门槛
	- 远方起动：
		- A 侧检测到正向故障，向 B 侧发送高频允许信号。
		- B 侧收信机收到 A 侧的信号。
		- B 侧保护装置进行内部逻辑判断：虽然我的电流启动元件没动，**但我也没有检测到反向故障**（即故障肯定不在我背后）。
		- 既然 A 侧发来了求救信号，我又确认背后没故障，B 侧的高频发信机就会被**远方起动**，自动向 A 侧“回发”一个允许信号（Echo）。
		- A 侧收到 B 侧的回信，逻辑满足（自己正向动作 + 收到对侧允许），**A 侧瞬间跳闸**。
