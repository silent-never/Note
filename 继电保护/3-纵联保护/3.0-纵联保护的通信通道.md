纵联定义：沿被保护元件的纵向，从元件的一端到另一端
横联定义：不是比较线路两端，而是比较同一端的两条平行线路上的电流。
- 通道类型
	- 导引线：
		- 有铠装屏蔽的通信线缆
		- 只应用于很短的重要输电线路，投资高，15~20km
	- 电力线载波：
		- 相-地通道：能量损耗大，特殊天气下衰减增大，收信功率过低将退出运行
		- 相-相通道：能量损耗较小，占用两根导线
		- 阻波器：对高频传输信号阻抗很大，避免信号传输到相邻线路去
		- 结合电容器：使高频收发机与工频线路绝缘
		- 联接滤波器：过滤工频
		- 高频收发机
	- 微波：
		- 微波通信能量很大
	- 光纤
- 原理
	- 方向比较式
	- 电流差动式
