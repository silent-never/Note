
## 一、 无功功率平衡基础

电力系统无功功率的平衡原则是：**“分层分区，就地平衡”**。 无功平衡基本方程式：

$$Q_{GC} = Q_{LD} + Q_L$$

_(电源总无功 = 负荷总无功 + 网络损耗)_

### 1. 无功负荷与网络损耗 ($Q_{LD} + Q_L$)

- **异步电动机：** 消耗励磁功率（系统无功负荷的主要部分）。
    
- **变压器无功损耗：** 包含励磁损耗和漏抗损耗。
    
- **输电线路无功损耗：** 串联电抗无功损耗（消耗） + 并联电纳充电无功（产生）。
    

### 2. 无功电源 ($Q_{GC}$)

- 发电机
    
- 同步调相机
    
- 静电电容器
    
- 静止无功补偿器 (SVC)
    
- 静止无功发生器 (SVG)
    

## 二、 电源无功特性与电压稳定

根据单端供电系统的功率特性，送到负荷节点的功率方程为：

$$\left. \begin{aligned} P_2 &= \frac{EU}{X}\sin\delta \\ Q_2 &= \frac{EU}{X}\cos\delta - \frac{U^2}{X} \end{aligned} \right\}$$

当电势 $E$ 恒定，有功需求 $P$ 也恒定时，消去相角 $\delta$，得到**无功需求与负荷电压的 V-Q 关系**：

$$Q = \sqrt{\left(\frac{EU}{X}\right)^2 - P^2} - \frac{U^2}{X}$$
![[无功平衡确定电压.png]]
**稳定平衡位置分析：** 由于一般负荷无功需求与电压成正相关，故系统的静态平衡交点必须取在**电源功率曲线的下降区域**时才稳定：

- **受扰动态过程：** 若电压升高，负荷无功需求增加，但受端能提供的无功反而产生缺额；负荷向电源索取无功增加，导致在线路上的电压降变大，迫使电压重新降低，恢复平衡。
    

## 三、 允许电压偏移及其危害

### 1. 正常运行允许电压偏移范围

- **> 35kV：** $\pm 10\%$
    
- **< 10kV：** $\pm 7\%$
    
- **220V 单相供电：** $+7\% \sim -10\%$
    

_(注：系统发生**故障**时，电流较大，允许电压偏移较正常时多放宽_ $5\%$_)_

### 2. 电压偏移的危害

- **对负荷的影响：** * 异步电动机的转矩与端电压的平方 ($U^2$) 成正比。电压降低会导致转差增大，定子电流增大，发热严重增加。
    
    - 电热设备出力与电压平方成正比，电压降低导致出力严重不足。
        
- **对电力系统的影响：**
    
    - 电压降低会导致输送相同功率下的电流增加，从而增加网损，严重时危及系统稳定性。
        
    - 电压过高会使设备绝缘受到损害，并增加线路的电晕损耗。
        

## 四、 中枢点电压管理

系统的负荷通常通过一些主要的供电点（**中枢点**）来供应。 **常见中枢点：**

1. 区域性水、火电厂的高压母线
    
2. 枢纽变电所的二次母线
    
3. 有大量地方负荷的发电机电压母线
    

### 1. 中枢点允许电压范围的确定

通过计算负荷调压要求与网损压降，来确定中枢点的电压约束范围。

- 负荷 $S_i$ 允许电压范围：$U_{i.\min} \sim U_{i.\max}$
    
- 网损压降 $\Delta U_{oi}$：$\Delta U_{oi.\min} \sim \Delta U_{oi.\max}$ _(注意：此处的 min, max 都是指系统运行方式，即负荷大小，而不是指网损的绝对数值大小)_
    

为了同时满足中枢点下属**多个负荷**的电压需求，中枢点的允许调压范围会变小，必须满足：

$$\max_i [U_{i.\min} + \Delta U_{oi.\min}] \leq U_{o} \leq \min_i [U_{i.\max} + \Delta U_{oi.\max}]$$

### 2. 三种调压方式

- **逆调压：** 在大负荷时，网络中的电流大，线路电压降落大，此时**人为提高**中枢点电压以抵消线路压降；在最小负荷时，网络电压降落小，此时**人为降低**中枢点电压。
    
- **顺调压：** 由于中枢点电压等于发电机母线电压减去输电线路压降，自然情况为负荷大时电压下降，负荷小时电压升高。顺调压只需略微控制：最大负荷时 $U_{o.\min} \geq 102.5\% U_N$；最小负荷时 $U_{o.\max} \leq 107.5\% U_N$。
    
- **常调压：** 介于上述两种调压方式之间，保持电压基本恒定。
    

## 五、 电压调整的具体措施

**压降分析基础：** 压降纵分量公式：$\Delta U = \frac{PR + QX}{U}$

- **低压电力网：** 电阻 $R$ 较大，$\frac{PR}{U}$ 占主导。
    
- **高压电力网：** 电抗 $X$ 较大，$\frac{QX}{U}$ 占主导。
    

### 措施 1：发电机调压

- **方式：** 利用自动励磁装置调整发电机输出电压。
    
- **特点：** 不需要增加额外费用。适用于满足近处地方负荷电压需求（逆调压标准：$U_{G.\max} = 105\%U_N, U_{G.\min} = U_N$）。
    
- **问题与局限：**
    
    1. 对于线路较长、供电范围大、有多级变压的系统，不同运行方式下压降变化幅度过大，单靠发电机调压无法满足需求。
        
    2. 发电机母线电压升高，需要发电机输出更多的无功，要求电厂有充裕的无功储备。
        
    3. 并列运行的电厂中，调整某厂电压将影响无功功率的重新分配。 _(结论：在大型电力系统中，发电机调压一般只作为一种辅助性调压措施。)_
        

### 措施 2：变压器变比调压

利用改变变比 $k = \frac{U_1}{U_2}$ 来调压。前提是系统无功供应比较充足。 二次侧电压公式：

$$U_2 = \frac{1}{k}(U_1 - \Delta U_{1T}) = \frac{1}{k} \left( U_1 - \frac{P_1 R + Q_1 X}{U_1} \right)$$

为了满足二次侧电压约束 $U_{2.\min} \leq U_2 \leq U_{2.\max}$，需满足：

$$\begin{aligned} \frac{1}{k} \left( U_1 - \frac{P_1 R + Q_1 X}{U_1} \right)\Bigg|_{U_{1.\min}, S_{\max}} &\geq U_{2.\min} \\ \frac{1}{k} \left( U_1 - \frac{P_1 R + Q_1 X}{U_1} \right)\Bigg|_{U_{1.\max}, S_{\min}} &\leq U_{2.\max} \end{aligned}$$

解出变比范围：$k_{\min} \leq k \leq k_{\max}$。 选择中心抽头：

$$k_{nol} = \frac{k_{\min} + k_{\max}}{2}$$

_(注意：降压变压器前方的损耗为“输电线路+变压器”；升压变压器损耗只有“变压器”自身。)_

### 措施 3：无功功率补偿调压

由于无功功率传送会引起电压降落和有功损耗，应遵循“就地平衡”原则在负荷侧提供无功补偿。 忽略压降横分量，未补偿时送端电压：$U_1 = U_2 + \frac{P_2 R + Q_2 X}{U_2}$ 增加补偿 $Q_c$ 后，末端电压变为 $U_2'$：$U_1 = U_2' + \frac{P_2 R + (Q_2 - Q_c)X}{U_2'}$ 由此可得将电压提升至 $U_2'$ 的补偿需求 $Q_c$：

$$Q_c = \frac{U_2'}{X} \left[ (U_2' - U_2) + \left( \frac{P_2 R + Q_2 X}{U_2'} - \frac{P_2 R + Q_2 X}{U_2} \right) \right]$$

由于括号内第二项数值较小，通常略去，简化为：

$$Q_c \approx \frac{U_2'}{X} (U_2' - U_2)$$

_(注意：若存在变压器，考虑高低压变比_ $k = \frac{U_1}{U_2}$_，上述公式中变量皆为归算到高压侧的数值，实际计算时代入归算后的目标电压_ $U_2' = U_{2.low}'$_)_

#### ① 负荷并联补偿设备

并联补偿主要改变 $\frac{QX}{U}$ 中的 $Q$。

1. **静电电容器：**
    
    - 大负荷时电压降低（需投入），小负荷时电压偏高（需切除）。
        
    - 为了充分利用补偿容量，应在最大负荷时投入所有电容器，最小负荷时全部退出。按此原则结合变压器分接头联合确定补偿容量。
        
2. **同步调相机：**
    
    - **过励磁：** 发出感性无功使电压升高。适用于最大负荷，投入额定容量。
        
    - **欠励磁：** 吸收感性无功使电压降低。适用于最小负荷，一般投入额定容量的 50%~65%（设此比例为 $\alpha$）。 联立方程求解容量需求和比例：
    $$\begin{aligned} Q_c &= \frac{U_{2.\max}'}{X}(U_{2.\max}' - U_{2.\max}) \\ -\alpha Q_c &= \frac{U_{2.\min}'}{X}(U_{2.\min}' - U_{2.\min}) \end{aligned}$$
    解得：
    
    $$-\alpha = \frac{U_{2.\min}' (U_{2.\min}' - U_{2.\min})}{U_{2.\max}' (U_{2.\max}' - U_{2.\max})}$$

#### 措施 4：线路串联电容补偿

串联补偿主要减小线路等效电抗 $X$ 变为 $X_L - X_C$。

- **补偿度：** $k_c = \frac{X_C}{X_L}$
    
- **优点：** 串联电容产生的压升 $\frac{QX_C}{U}$ 在负荷大时增大，负荷小时减小，这种自适应特性与调压的期望要求完美一致。
    
- **问题：** 功率传输方向不恒定时可能引起过电压，因此串联电容一般放置在线路中间。
    
- **应用场景：** 主要用于 10kV 或 35kV 且负荷波动大而频繁、功率因数很低的配电线路。
