- **电压降落 (Voltage Drop)**：元件首端与末端电压的**相量差**：$\Delta \dot{U}+\delta \dot{U} = \dot{U}_1 - \dot{U}_2$
	- 纵分量：$\Delta \dot{U}\; \parallel \dot{U}$
	- 横分量：$\delta \dot{U}\;\perp \dot{U}$
	- 注意到，对于 $\dot{U}_1$ 和 $\dot{U}_2$ 的纵分量和横分量是不一样的
- 取电压 $\dot{U}_2=U_2\angle 0 ^\circ, \quad \dot U_1=U_1\angle \delta ,\quad \dot I=I\angle -\varphi_2$
	- 有$$\dot{U}_1 - \dot{U}_2 = \Delta U_2+j\;\delta U_2=(R + \mathrm{j}X)\dot{I}$$
