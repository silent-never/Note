### 一、保护原理
- 电流保护：依据电流幅值或电流矢量和实现故障判别
- 距离保护：依据测量故障点至保护安装处的阻抗实现故障判别
- 方向保护：依据功率方向实现故障判别
- 纵联保护：利用通道交换线路两端电气量信息实现故障判别
- 瓦斯保护：依据变压器内部气体积累、压力变化等非电量信号实现故障监测
### 二、辅助功能
- 自动重合闸：线路跳闸后自动尝试重新合闸，以恢复瞬时性故障后的供电，减少瞬时故障危害。
### 三、保护对象
- 线路：配置电流、距离、纵联等保护，快速切除线路故障。
- 变压器：配置差动、瓦斯、过流等保护，反映内部及外部故障。
- 发电机：配置差动、过电压、负序电流等保护，防止定子、转子等损坏。
- 母线：配置比率差动保护，快速清除母线短路故障。
- 其他（电动机、电容器等）：配置过流、过压、欠压等保护，保障设备安全运行。
### 四、其他定义
-  后备保护：
	- 主保护不能正常快速切除故障情况下
		- 保护装置拒动
		- 保护回路中其他环节损坏
		- 断路器拒动
	- 迅速启动切断故障
	- 近后备：
		- 当主保护拒动时，由本元件（线路、变压器等）的另一套不同原理的保护或本断路器相关的失灵保护来实现后备。
		- 优点：
			- 动作速度快（一般延时短，如断路器失灵保护延时150~200ms）——同一断路器
			- 停电范围小（仅跳开本元件或本级母线，不波及相邻元件）——对系统稳定性影响小
			- 灵敏度高——满足高压电网要求
		- 缺点：
			- 对直流电源有要求，当变电所直流系统故障时与主保护一同失效
			- 无法后备保护断路器故障
	- 远后备：
		- 优点：
			- 保护范围覆盖所有下级电力元件，解决任何原因的故障元件切除
		- 缺点：
			- 动作时间较长（需与下级保护配合，一般为0.5~1.0s以上）
			- 停电范围大（会切除上级母线或更多线路）
			- 在复杂环网中，灵敏度和选择性配合困难
